clone - cloning the repo on local machine
>> git clone <repo-link>

status- display the state of the code
>> git status (show the status of modified file or new file add)

four type of status 
1.untracked-new file that git does not track
2.modified - changed commit changes in file
3.staged - file is ready to be committed
4.unmodified - unchanged



****Add and commit commands***

ADD - add new file or change the existing file in your working directory to the git staging name
Git add <file name>


COMMIT – it is the record of the changes
Git commit –m “your commit massage”


Push command 
Push – upload local repo content to remote repo(github.com)
Git push origin main
Main – is the branch
